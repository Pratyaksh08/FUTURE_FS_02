# 🚀 Mini CRM - Client Lead Management System

## 📌 Project Overview
A full-stack CRM application to manage client leads efficiently.  
Users can add, update, delete, and search leads in real-time.

---

## 🛠 Tech Stack
- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB
- Styling: Tailwind CSS

---

## ✨ Features
- ➕ Add new leads
- 🔄 Update lead status (New / Contacted / Converted)
- ❌ Delete leads
- 🔍 Search leads instantly
- 📊 Clean and responsive UI
